Test Google Sheets Integration

This page will help you test if your Google Sheets integration is working properly.

Instructions:

  1. Make sure you have a Google Sheet open
  2. Copy the Google Apps Script code from google-apps-script.js
  3. Go to Extensions → Apps Script in your Google Sheet
  4. Paste the code and save
  5. Deploy as a web app with "Anyone" access
  6. Update the script URL in the form below
  7. Test the submission
